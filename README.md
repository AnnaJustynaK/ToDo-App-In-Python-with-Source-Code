# ToDo-GUI-Python
Build a simple To-Do GUI application using python

About the system
This todo app is designed to create your to-do list and delete the list as per the user wish. Here, the module used is Tkinter()-It is a standard Python interface to the Tk GUI toolkit shipped with Python. Python with tkinter outputs the fastest and easiest way to create the GUI applications.
